# mmtc-webapp-ui

Stack:
- Vue 3
- Nuxt 3
- NuxtUI 4

## Development

### Requirements

- [Node](https://nodejs.org/en/download)
  - Tested to work with node v22.19.0
- [PNPM](https://get.pnpm.io)
  - Tested to work with pnpm 10.18.1

## Building

```sh
# first time only:
pnpm install

# to start a development server with hot reloading:
pnpm dev

# to generate static bundle:
npx nuxt generate 
```
